7.1.- Gestión de servicios en modo comando.

Vamos a ver como parar y arrancar los servicios desde línea de comandos en un sistema Windows 7.

Tenemos varias opciones para parar y arrancar servicios:

  • Comando net.
  • Comando sc.
  • Comandos Netsvc.exe e Instsrv.exe para trabajar con un equipo remoto.

Comando NET

NET es una utilidad del sistema operativo windows Win2k y XP que sirve para acceder a servicios de red. Permite hacer muchísimas cosas y entre ellas está la posibilidad de arrancar y parar un servicio.

  • Para ver los servicios iniciados: Net start
  • Para iniciar un servicio: Net start <nombre_del_servicio>
  • Para detener un servicio: Net stop <nombre_del_servicio>
  • Para pausar un servicio: Net pause <nombre_del_servicio>
  • Para reanudar un servicio: Net continue <nombre_del_servicio>

Ejemplo:

 
>net stop "Actualizaciones automáticas"
El servicio de Actualizaciones automáticas está deteniéndose.
El servicio de Actualizaciones automáticas fue detenido con éxito.

 >net start "Actualizaciones automáticas"
 El servicio de Actualizaciones automáticas está iniciándose.
 El servicio de Actualizaciones automáticas se ha iniciado con éxito.

 

 

NOTA: Con NET se puede usar tanto el nombre del servicio como el nombre para mostrar (“Actualizaciones Automáticas” y “wuauserv”).

Comando SC

Sc.exe, que se incluye en el Kit de recursos de Windows XP y viene de serie en Windows 7, implementa llamadas a todas las funciones de la interfaz de programación de aplicaciones (API) de control de servicios de Windows a través de la línea de comandos.

Con este comando se puede mostrar el estado de los servicios, pararlos, arrancarlos, recuperar los valores almacenados en los campos de estructura de estado, etc.

También permite especificar el nombre de un equipo remoto para trabajar con los servicios de forma remota.

La sintaxis del comando es:

sc [nombreDeServidor] Comando nombreDeServicio [nombreDeOpción = valorDeOpción...]

Así tenemos:

  • Para iniciar un servicio: sc start <nombre_del_servicio>
  • Para detener un servicio: sc stop <nombre_del_servicio>
  • Para consultar los servicios corriendo: sc query

Ejemplo:

 
>sc start wuauserv
